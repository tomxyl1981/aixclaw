# 那耶村夏令营"变形计"版设计

> 琳燊提议：加入支教+交换体验 | 2026-05-22

---

## 核心思路

```
城市孩子来那耶村 → 体验乡村生活 + 支教当地孩子
当地孩子加入夏令营 → 免费学AI + 展示壮族文化
双向交换 → 城市孩子学会感恩，乡村孩子看见未来
```

---

## 一、支教模块设计

### 1.1 城市孩子教什么

| 课程 | 内容 | 时长 |
|------|------|------|
| 英语启蒙 | 简单英语对话+英文歌 | 2小时 |
| AI工具入门 | 怎么用AI画画、做音乐 | 2小时 |
| 世界见闻 | 分享城市生活、旅行经历 | 1小时 |
| 读书分享 | 推荐一本自己喜欢的书 | 1小时 |

### 1.2 乡村孩子教什么

| 课程 | 内容 | 时长 |
|------|------|------|
| 壮族铜鼓 | 铜鼓历史+简单敲法 | 2小时 |
| 梯田农耕 | 插秧/收割+农具使用 | 2小时 |
| 壮语课堂 | 简单壮语对话+壮族歌曲 | 1小时 |
| 草药辨认 | 山里常见草药+用途 | 1小时 |

### 1.3 双向收益

| 维度 | 城市孩子 | 乡村孩子 |
|------|----------|----------|
| 技能 | 学会农耕+民族才艺 | 学会AI工具+英语 |
| 视野 | 看见乡村真实生活 | 看见城市和AI世界 |
| 品格 | 学会感恩和尊重 | 获得自信和希望 |
| 社交 | 结交乡村伙伴 | 结交城市伙伴 |
| 履历 | "支教经历"写入证书 | "AI创作认证"写入证书 |

## 二、"变形计"式体验设计

### 2.1 Day1-2：城市孩子"变形"

| 时间 | 活动 | 设计意图 |
|------|------|----------|
| 早起 | 6:00起床，跟村民去梯田 | 体验真实农活 |
| 上午 | 插秧/除草（当地孩子带队） | 角色反转，乡村孩子当老师 |
| 中午 | 吃那耶米饭+壮家菜 | 感受简单食物的珍贵 |
| 下午 | 割草/喂鸡/挑水 | 体力劳动，理解辛苦 |
| 晚上 | 无手机，听村长讲故事 | 断网体验，回归人与人 |

**关键设计**：前2天没收手机，让孩子真正"变形"

### 2.2 Day3-4：双向课堂

| 上午 | 下午 |
|------|------|
| 城市孩子教AI工具 | 乡村孩子教壮语+铜鼓 |
| 城市孩子教英语歌 | 乡村孩子带路认草药 |
| 双方一起做梯田调查 | 双方一起画梯田写生 |

**关键设计**：不再是"城里人来教乡下人"，而是平等交换

### 2.3 Day5-6：共同创作

| 项目 | 形式 | 合作方式 |
|------|------|----------|
| "我的那耶"短片 | 城市孩子拍摄+AI剪辑 | 乡村孩子出镜+讲述 |
| 壮族民歌新编 | AI音乐+传统唱法 | 乡村孩子唱+城市孩子编曲 |
| 梯田故事绘本 | AI绘画+手写文字 | 共同创作 |
| "变形日记" | 每天写一段感受 | 交换日记 |

### 2.7 Day7：成果展示

| 环节 | 内容 |
|------|------|
| 10:00 | 支教成果汇报（城市孩子展示乡村课堂成果） |
| 11:00 | 文化展示（乡村孩子展示壮语+铜鼓+AI作品） |
| 14:00 | 共同创作作品展映 |
| 15:00 | "变形对话"——城市孩子vs乡村孩子对谈 |
| 16:00 | 颁发证书+交换礼物 |
| 17:00 | 告别仪式 |
