## 一、《隐没的战象》的教育意义

### 1.1 作品背景

| 维度 | 说明 |
|------|------|
| 作者 | 梁越，百万流量作家，那耶村名誉村长 |
| 类型 | 长篇历史小说，50万字 |
| 内容 | 北宋时期侬智高事件，南方民族史 |
| 影响 | 获中国作家网连载，流量一千几百万 |

### 1.2 对孩子的五大教育意义

| 教育维度 | 核心价值 | 孩子能学到什么 |
|----------|----------|----------------|
| **历史观** | 打破中原中心叙事 | 中国不止有中原历史，南方少数民族也有自己的英雄和故事。孩子学会"历史不只一种视角" |
| **正义观** | 理解"正义性"的含义 | 侬智高为什么起兵？书中揭示了他行动的正义性。孩子学会不盲从标签，学会追问"为什么" |
| **文化认同** | 看见自己的祖先故事 | 南方少数民族孩子第一次看到自己的历史被正面书写。文化认同感是身份自信的基础 |
| **全球视野** | 理解东南亚格局 | 侬智高事件涉及越南、东南亚。孩子学会中国历史与世界历史的关联 |
| **文学素养** | 50万字的叙事结构 | 孩子学习"史诗级"作品的叙事手法，比碎片化短视频更有深度 |

### 1.3 村长亲讲的价值

| 维度 | 说明 |
|------|------|
| 不是教科书 | 村长站在梯田边讲，是"活的历史" |
| 不是45分钟讲座 | 7天×24小时随时可以对话 |
| 不是"嘉宾" | 村长就住在这儿，孩子随时可以找到他 |
| 不是单向输出 | 孩子可以提问、质疑、讨论 |

**一句话：在上海听作家讲座是45分钟，在那耶村是7天和作家一起生活。**

## 二、艺术+技术培训项目设计

### 核心理念

```
艺术 = 感受力 + 表达力
技术 = AI工具 + 实现手段
那耶村 = 实景素材库 + 文化密码
```

**目标：让孩子在万亩梯田里，用AI工具创作艺术作品**

---

### 项目一：战象IP·视觉艺术项目

#### 项目名称
**"战象归来"——AI绘画+版画风格创作**

#### 项目设计

| 步骤 | 内容 | 技术 | 艺术 |
|------|------|------|------|
| Step1 | 村长讲战象故事（1小时） | - | 听觉感受 |
| Step2 | 梯田实景观察+拍摄（2小时） | 手机摄影 | 视觉观察 |
| Step3 | 学习梁越版画风格（1小时） | Midjourney风格参考 | 艺术史 |
| Step4 | AI生成战象形象（2小时） | Midjourney/SD | 数字创作 |
| Step5 | 手绘修改+版画叠加（2小时） | Procreate | 手工质感 |
| Step6 | 梯田背景合成（1小时） | PS/Canva | 场景构建 |

#### 输出成果

| 成果 | 说明 |
|------|------|
| 战象角色设计图 | AI生成+手绘修改 |
| 梯田战象场景图 | 战象+万亩梯田合成 |
| 版画风格战象 | 梁越《痕》艺术展风格迁移 |
| 展出机会 | 作品可入选那耶村电影节展映 |

#### 艺术教育点

| 维度 | 学习内容 |
|------|----------|
| 观察力 | 学会观察梯田的光影、层次、空间 |
| 感受力 | 听故事→转化成视觉想象 |
| 表达力 | 用画面表达"战象守护那耶村"的意象 |
| 风格理解 | 学习版画风格的艺术语言 |
| 技术应用 | AI不是替代创作，是放大创作 |

## 三、艺术+技术教育理念

### 3.1 核心公式

```
艺术感受力 + AI技术工具 + 那耶实景素材 = 原创艺术作品
```

### 3.2 不是技术取代艺术

| 错误理解 | 正确理解 |
|----------|----------|
| AI帮孩子画画 | 孩子感受→AI放大→孩子修改 |
| 技术替代创作 | 技术是工具，感受是核心 |
| 学会软件就够了 | 学会感受+学会工具 |
| 作品是AI生成的 | 作品是孩子的感受+AI的实现 |

### 3.3 教育目标

| 目标 | 说明 |
|------|------|
| 感受力 | 能观察、能感受、能想象 |
| 表达力 | 能用画面/音乐/影像表达 |
| 技术力 | 能用AI工具实现想象 |
| 项目力 | 能完成一个完整创作项目 |
| 展现力 | 作品可以展出、上架、传播 |
