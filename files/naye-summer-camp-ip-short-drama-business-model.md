## 一、《霍去病》AI短剧案例核查（2026年3月）

### 1.1 案例背景

| 维度 | 说明 |
|------|------|
| 作品 | 《霍去病》AI短剧 |
| 导演 | 杨涵涵（专业影视导演） |
| 内容 | 霍去病漠北之战历史题材 |
| 发布时间 | 2026年2月16日 |
| 平台 | 抖音、B站、海外多平台 |

### 1.2 数据核查（传闻 vs 实际）

| 维度 | 最初传闻 | 导演澄清 |
|------|----------|----------|
| **播放量** | 5亿+ | ✅ **确认（全网累计）** |
| **成本** | 3000元 | ❌ **3000元只是算力成本，不含人力** |
| **团队** | 3人 | ❌ **实际近20人团队** |
| **时长** | 80集 | ❌ **实际4分钟+6分钟两条短片** |
| **制作时间** | 5天 | ❌ **48小时纯工作时间（约6天）** |

### 1.3 导演杨涵涵澄清要点

| 澄清内容 | 说明 |
|----------|------|
| 成本构成 | 3000元仅为AI算力支出，不含团队人力成本 |
| 团队规模 | 近20人（创意策划+内容审核+细节优化） |
| 实际时长 | 4分钟+6分钟两个版本，非80集 |
| 历史考据 | 前期资料整理+历史考证用时2天 |
| 质量标准 | 追求"影视级别"，剧本设计扎实 |

### 1.4 行业对比

| 维度 | 传统短剧 | AI短剧《霍去病》 |
|------|----------|------------------|
| 单集成本 | 3-8万元 | 千元级（算力） |
| 80集制作周期 | 数月 | 48小时 |
| 战场特效 | 1天 | 1小时（AI） |
| 团队规模 | 50-100人 | 20人 |

### 1.5 红果平台AI短剧政策（2026）

| 政策 | 说明 |
|------|------|
| AI仿真人短剧分成系数 | **60（漫剧最高）** |
| 真人短剧保底 | 全面取消（转向精品化） |
| AI漫剧市场规模 | **240亿元（2026年预测）** |
| AI真人短剧市场 | **120亿元（2025年，同比+300%）** |

### 1.6 案例启示

| 启示 | 对那耶村夏令营的意义 |
|------|----------------------|
| AI能降低门槛 | 学生+AI工具可完成专业级短剧 |
| 播放量真实 | 5亿播放量验证市场需求 |
| 专业指导重要 | 导演历史考据2天→导师指导必要性 |
| 成本效率突破 | 传统5万/集→AI千元级 |
| 独家IP优势 | 公共IP（霍去病）vs 独家IP（战象） |

## 二、那耶村夏令营短剧项目设计

### 2.1 核心定位

```
学生 + 作家村长梁越导师 + 《隐没的战象》IP + AI创作工具 = 红果平台短剧变现
```

### 2.2 与《霍去病》案例对比

|------|----------------|------------------|
| 创作者 | 专业导演20人团队 | **学生+双导师** |

| 门槛 | 专业门槛高 | **导师降低学生门槛** |

## 三、导师价值与收费

### 3.1 双导师配置

| 导师 | 身份 | 价值 |
|------|------|------|
| **梁越** | 作家、村长、《隐没的战象》作者 | IP授权+故事指导+历史考据 |
| **Jane** | AIX创始人、AI创作专家 | AI工具方法+平台运营+变现策略 |

### 3.2 导师价值对比

| 维度 | 传统夏令营 | 那耶村短剧夏令营 |
|------|-----------|------------------|
| 老师 | 普通培训师 | 百万流量作家+AI创业创始人 |

| 产出 | 做作业 | 上架变现的短剧 |
| 收益 | 无后续收益 | 作品持续产生收入 |
| IP授权 | 无 | 独家IP使用权+分成权 |

### 3.3 收费设计

| 项目 | 传统夏令营 | 那耶村短剧夏令营 |
|------|-----------|------------------|
| 学费 | 3000-8000元 | **10,000元** |
| 食宿 | 另算 | 含在学费内 |

## 四、短剧创作流程（7天）

### 4.1 课程安排

| 天数 | 上午 | 下午 | 导师参与 |
|------|------|------|----------|
| Day1 | 村长梁越讲《隐没的战象》核心情节 | 梯田实景观察+素材采集 | 梁越主讲 |
| Day2 | Jane讲AI短剧创作方法论 | AI工具入门（MJ/SD/Runway） | Jane主讲 |
| Day3 | 学生分组+剧本设计 | 分镜设计（6-8关键画面） | 双导师指导 |
| Day4 | AI生成画面 | AI视频生成 | 导师技术支持 |
| Day5 | 剪辑+配音 | 字幕+配乐+上架准备 | 导师审核 |
| Day6 | 红果/抖音上传 | 数据分析+运营建议 | 导师运营指导 |
| Day7 | 成果展示+分成机制讲解 | 结营+证书颁发 | 双导师总结 |

### 4.2 输出成果

| 成果 | 说明 |
|------|------|
| 10集×3分钟短剧 | 学生原创战象短剧 |
| 分镜脚本 | 学生原创叙事设计 |
| 红果/抖音上架 | 直接变现渠道 |
| 收益分成 | 学生50%+IP池20%+平台20%+基金会10% |
