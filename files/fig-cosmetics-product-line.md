## 一、无花果的美容价值

| 成分 | 功效 | 适用肤质 |
|------|------|----------|
| 花青素 | 抗氧化、抗衰老 | 熟龄肌 |

| 无花果酶 | 分解老化角质 | 所有肤质 |
| 类黄酮 | 抗炎、修复 | 痘痘肌 |

## 四、生产模式

### 方案A：OEM代工（推荐）
| 优势 | 说明 |
|------|------|
| 启动快 | 3-6个月上市 |
| 合规成熟 | 工厂已有备案资质 |
| 成本可控 | 小批量起订 |
| 品牌自主 | 那耶村品牌 + 无花果原料 |

### 合作方选择
| 类型 | 代表企业 | 起订量 |
|------|----------|--------|
| 大厂代工 | 科丝美诗、莹特丽 | 5000件起 |
| 中小代工 | 广州/上海代工厂 | 1000件起 |
| 天然护肤专项 | 云南植物提取工厂 | 500件起 |

| 产品 | 人民币定价 | Coin Hour定价 |

**闭环逻辑**：
```
那耶村种植无花果 → 提取果汁/果干 → 供应OEM工厂
        ↓
品牌销售战象美肌 → Coin Hour结算
        ↓
收益分配 → 村集体分红 + 农户原料收入
```
