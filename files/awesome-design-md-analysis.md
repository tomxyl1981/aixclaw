## 一、项目核心价值

### 问题：AI生成UI不一致

| 问题 | 说明 |
|------|------|
| 配色混乱 | 每次 AI 生成的颜色都不一样 |
| 字体不一致 | 同一项目出现多种字体 |
| 组件风格差异 | 按钮样式五花八门 |
| 品牌识别度低 | 看不出是哪个品牌 |

### 解决方案：DESIGN.md

```
品牌设计系统 → 整理成 DESIGN.md → AI 读取规则 → 生成一致 UI
```

```markdown
# Stripe Design System

## 配色

- 标题: sans-serif, 600 weight

## 间距
- 小: 8px
- 中: 16px
- 大: 32px
- 超大: 64px
```

## 七、那耶村品牌 DESIGN.md（建议）

```markdown
# 那耶村设计系统

## 配色
- 主色: #2D5016（梯田绿）
- 辅色: #F4A460（战象色）
- 背景: #F5F5DC（稻穗黄）
- 强调色: #8B4513（壮乡土）
- 文字: #1A1A1A / #666666

## 字体
- 标题: Noto Serif SC（衬线，文化感）
- 正文: Noto Sans SC（无衬线，易读）
- 英文: Inter

## 组件
- 按钮: 圆角12px, 梯田绿渐变
- 卡片: 白色/米色, 阴影柔和
- 图片: 圆角16px, 梯田/云海为主

## 图标风格
- 手绘风格（壮族元素）
- 战象 IP 插画
- 梯田线条

## 情绪
- 温暖、自然、宁静
- 千年古村的厚重感
- 艺术家的精致感
```

