## 四、Graphify实战教程

### 4.1 安装

```bash
# pip安装（注意包名graphifyy，两个y）
pip install graphifyy && graphify install

# 验证安装
graphify --version
```

### 4.2 基础操作

```bash
# 创建素材文件夹
mkdir -p ~/naye-village-materials
cd ~/naye-village-materials

# 放入素材
# ├── terraces/（梯田照片）
# ├── folk-songs/（民歌录音）
# ├── stories/（梁越故事笔记）
# └── culture/（壮族文化资料）

# 生成知识图谱
graphify .

# 查看图谱统计
graphify stats

# 可视化
graphify visualize
```

### 4.3 查询操作

```bash
# 查询：那耶村的核心文化密码是什么？
graphify query "那耶村的核心文化密码是什么？"

# 查询：象耕鸟耘和铜鼓的关系
graphify path "象耕鸟耘" "铜鼓"

# 解释：壮族民歌
graphify explain "壮族民歌"
```

### 4.4 Wiki生成

```bash
# 生成Wikipedia风格文章
graphify . --wiki

# 输出：那耶村百科.md
```

### 4.5 增量更新

```bash
# 新增素材后
graphify . --update

# 只处理新增/修改文件
```
