## 三、录音流程

### 3.1 录音前准备（Day -1）

**设备检查清单**：
- [ ] 所有电池充满
- [ ] 存储卡格式化（exFAT）
- [ ] 测试所有话筒
- [ ] 校准录音机电平
- [ ] 准备场记表模板

**歌师对接**：
- [ ] 确认歌师姓名、联系方式
- [ ] 说明录音用途（需签署授权书）
- [ ] 约定时间与地点
- [ ] 准备茶水/报酬

### 3.2 录音当天流程

**Step 1 场地勘察（30分钟）**
```
1. 选择安静空间（避开公路、施工）
2. 测试自然混响（拍掌听回声）
3. 确定录音位置（歌手/乐器 vs 话筒距离）
4. 布置简易吸音（窗帘、毯子减少反射）
```

**Step 2 设备架设（20分钟）**
```
1. 录音机放置稳固桌面
2. 话筒架设：乐器1.5-2米，人声0.5-1米
3. 指向性检查：心形话筒对准声源
4. 电平测试：峰值-12dB至-6dB，避免失真
```

**Step 3 正式录音**

| 步骤 | 操作 | 场记记录 |
|------|------|----------|
| 打板 | "第X首，[曲名]，[歌师名]，第1条" | 时间码：00:00:00 |
| 录音 | 保持安静，不要移动 | 观察电平表 |
| 结束 | 等待2秒静默再停止 | 标记结束时间 |
| 备份 | 立即双备份 | 检查文件完整性 |

### 3.3 录音后工作（当天完成）

**文件整理**：
```
文件夹结构：
Naye_Recording_20260522/
├── 原始音频/
│   ├── Zhuang/
│   │   ├── Tonggu/
│   │   ├── Tianqin/
│   │   └── Shange/
│   └── Miao/
│       ├── Lusheng/
│       ├── Mangtong/
│       └── Feige/
├── 编辑后/
├── 场记表/
└── 照片视频/
```

**命名规范**：
```

示例：

（Z=壮族，Take01=第1条）
```

## 六、质量控制

### 6.1 录音质量标准

| 指标 | 要求 | 检查方法 |
|------|------|----------|
| **采样率** | 48kHz / 96kHz | 录音机设置确认 |
| **位深** | 24bit | 录音机设置确认 |
| **电平** | 峰值-12dB至-6dB | 实时观察电平表 |
| **底噪** | 无电流声/风扇声 | 录音前静默测试 |
| **清晰度** | 无混响过大/过小 | 听感测试 |

### 6.2 常见问题解决

| 问题 | 原因 | 解决 |
|------|------|------|
| 电平过低 | 话筒距离远 | 靠近至1米内 |
| 电平过载 | 距离近/嗓门大 | 开启PAD衰减/拉远距离 |
| 环境噪音 | 户外录音 | 选择安静时段，使用指向性话筒 |
| 房间混响 | 空间空旷 | 增加吸音材料，选择小房间 |
| 歌手紧张 | 不适应录音 | 先聊天放松，录制多个Take |
