# 那耶村音乐节场地搭建技术方案

> 编制日期：2026-05-22
> 场地：那耶村梯田核心区

---

## 一、场地选址分析

### 1.1 推荐位置

| 要素 | 要求 | 那耶村条件 |
|------|------|-----------|
| 视野 | 梯田全景 | ✅ 万亩梯田观景台 |
| 声场 | 自然混响 | ✅ 山谷环绕 |
| 交通 | 便利到达 | ⚠️ 需规划临时道路 |
| 安全 | 疏散通道 | ⚠️ 需设计应急预案 |

### 1.2 场地布局

```
        【梯田景观】
             ↓
    ┌────────────────┐
    │   观演台(舞台)   │ ← 主表演区
    └────────────────┘
             ↓
    ┌────────────────┐
    │   VIP座位区     │ ← 50个席位
    └────────────────┘
             ↓
    ┌────────────────┐
    │   普通座位区     │ ← 150个席位
    └────────────────┘
             ↓
    【入口/售票处】
```

## 二、舞台搭建方案

### 2.1 舞台规格

| 参数 | 数值 | 备注 |
|------|------|------|
| 舞台面积 | 100㎡ (10m×10m) | 钢架结构 |
| 舞台高度 | 0.8-1.2m | 离地面高度 |
| 承重 | 500kg/㎡ | 满足设备+人员 |
| 防滑 | 铝合金防滑板 | 安全考虑 |

### 2.2 舞台结构

```
顶层：铝合金防滑板
      ↓
中层：钢架支撑结构
      ↓
底层：地面固定（膨胀螺栓）
```

### 2.3 舞台设备清单

| 设备 | 数量 | 功率 | 备注 |
|------|------|------|------|
| 主扩音响 | 4组 | 1000W/组 | JBL/Meyer |
| 返听音响 | 4个 | 300W/个 | 舞台返送 |
| 线阵列 | 2串 | - | 远场覆盖 |
| 调音台 | 1台 | - | 32路数字台 |

## 七、安全预案

### 7.1 疏散通道

- 主通道：宽度≥2m
- 备用通道：≥1.5m
- 疏散标识：荧光指示牌
- 应急照明：蓄电池灯

### 7.2 应急设备

| 设备 | 数量 | 位置 |
|------|------|------|
| 灭火器 | 4个 | 舞台+观众席 |
| 急救箱 | 1个 | 入口处 |
| 警戒线 | 100m | 周边区域 |
