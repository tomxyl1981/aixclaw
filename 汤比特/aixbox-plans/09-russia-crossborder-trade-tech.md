## 📋 完整交易流程

### 阶段1：商品发现

```
俄罗斯买家 (AIX Box)
    ↓
LBRTYnet Mesh 搜索 (AGORA:search)
    ↓
发现中国卖家商品
    ↓
查看商品详情 (AIX/CH计价)
```

**技术实现：**
- 借鉴Agora Market的商品列表机制
- 使用BUS消息 `AGORA:listing_new` 广播商品
- 支持AIX Coin Hour定价

---

### 阶段2：订单创建

```
买家发起交易请求
    ↓
创建托管合约 (多签名)
    ↓
锁定 Coin Hour 到托管地址
    ↓
卖家收到订单通知
```

**Coin Hour 托管机制：**
```json
{
  "trade_id": "RU-CN-2026040901",
  "buyer": "俄罗斯买家钱包地址",
  "seller": "中国卖家钱包地址",
  "arbiter": "AIX仲裁节点地址",
  "amount": "100000 CH",  // 约 $100
  "state": "escrow",      // 托管中
  "created": "2026-04-09T01:00:00Z"
}
```

**托管释放条件：**
- 买家确认收货 → 释放给卖家
- 争议 → 仲裁节点裁决
- 超时 → 自动释放

---

### 阶段3：物流追踪

```
卖家发货
    ↓
清关AI处理 (HS编码匹配)
    ↓
物流信息上链 (每节点更新)
    ↓
买家实时查看位置
```

**物流节点：**
| 节点 | AIX Box | 状态更新 |
|------|---------|----------|
| 中国仓库 | ✅ | 已发货 |
| 满洲里口岸 | ✅ | 清关中 |
| 俄罗斯边境 | ✅ | 已入境 |
| 买家地址 | ✅ | 已签收 |

**技术：**
- 每个物流节点运行AIX Box
- 通过LBRTYnet Mesh传输状态
- 状态签名上链防篡改

---

### 阶段4：清关处理

```
货物到达口岸
    ↓
清关AI自动处理
    ↓
HS编码匹配
    ↓
生成清关文件
    ↓
合规检查 → 通过/警告
```

**清关AI功能：**
1. **HS编码匹配**
   - 自动识别商品类型
   - 匹配正确的HS编码
   - 计算关税税率

2. **合规检查**
   - 制裁商品名单检查
   - 数量限制检查
   - 许可证要求检查

3. **文件生成**
   - 商业发票
   - 装箱单
   - 原产地证明

---

### 阶段5：支付结算

```
买家确认收货
    ↓
托管合约释放 Coin Hour
    ↓
卖家收到 CH
    ↓
可选：CH → AIX → 法币
```

**Coin Hour 作为传输燃料：**

| 角色 | 用途 | CH消耗 |
|------|------|--------|
| 买家 | 支付货款 | 100 CH |
| 卖家 | 网络传输费 | 5 CH |
| 货代 | 物流节点费 | 10 CH |
| 清关 | AI服务费 | 5 CH |
| **总计** | | **120 CH** |

## 💰 Coin Hour 经济模型

### 为什么用 Coin Hour？

| 对比项 | 法币 | USDT | AIX | Coin Hour |
|--------|------|------|-----|-----------|
| 跨境转账 | 慢/贵 | 中等 | 快 | 最快 |
| 手续费 | 高 | 中 | 低 | 极低 |
| 稳定性 | 高 | 高 | 低 | 相对稳定 |
| 铸造 | 无限 | 1:1 | 限量 | 每小时+1 |
| 销毁 | 无 | 无 | 无 | 转账销毁50% |

**Coin Hour 优势：**
- **无需交易所** — 钱包自动产生
- **低门槛** — 持有AIX即有CH
- **防通胀** — 转账销毁50%控制供应

### CH/USD 汇率机制

**预言机方案：**
```
每小时采样：
- LBank AIX/USD 价格
- 计算 CH 价值 = AIX价格 × 质押率
- 发布到链上预言机
```

**示例：**
```
AIX 价格 = $0.50
质押 100 AIX → 每小时产生 100 CH
CH 价值 ≈ $0.001/CH (估算)
```
