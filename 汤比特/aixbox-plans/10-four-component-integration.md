## 🎯 每个组件做什么？

### 1️⃣ AIX Box（硬件层）

**是什么：** 物理设备，RK3588芯片的小型服务器

**做什么：**
| 功能 | 说明 |
|------|------|
| 运行节点 | AIX全节点、Skywire节点、LBRTYnet节点 |
| AI计算 | 本地AI助手、清关AI |
| 存储 | 贸易数据、物流记录 |
| 通信 | Mesh网络路由器 |

**在贸易中的角色：**
- 中国卖家：放置在仓库/办公室
- 俄罗斯买家：放置在家中
- 物流节点：放置在口岸/货代

---

### 2️⃣ LBRTYnet（网络层）

**是什么：** 去中心化Mesh网络协议

**做什么：**
| 功能 | 说明 |
|------|------|
| 节点发现 | 找到其他AIX Box节点 |
| 消息传递 | 贸易信息、订单、物流状态 |
| 穿透打洞 | 绕过NAT/防火墙 |
| 离线通信 | 无互联网时Mesh互联 |

**在贸易中的角色：**
```
中国卖家 ────┐
            ├──→ LBRTYnet Mesh ──→ 俄罗斯买家
物流节点 ────┘
```

**为什么需要：**
- 俄罗斯部分地区网络差
- 避免中心化平台审查
- 断网也能本地通信

---

### 3️⃣ AIX（价值层）

**是什么：** 主代币，类似比特币

**做什么：**
| 功能 | 说明 |
|------|------|
| 价值存储 | 长期持有，保值增值 |
| 大额支付 | 批量订单、B2B交易 |
| 治理 | DAO投票、参数调整 |
| 产生Coin Hour | 持有AIX自动产生CH |

**在贸易中的角色：**
- 大额订单用AIX支付
- 卖家持有AIX获得CH收入
- 汇率锚定：AIX/USD/卢布

---

### 4️⃣ Coin Hour（燃料层）

**是什么：** 传输燃料，持有AIX自动产生

**做什么：**
| 功能 | 说明 |
|------|------|
| 网络传输费 | 支付Mesh网络带宽 |
| 小额支付 | 零件、样品订单 |
| 服务费 | 清关AI、仲裁服务 |
| 防垃圾 | 每次转账消耗CH |

**在贸易中的角色：**
```
订单支付：     100 CH
网络传输费：   5 CH
清关AI服务：   5 CH
物流节点费：   10 CH
─────────────────
总计：        120 CH
```

**优势：**
- 无需购买：持有AIX自动产生
- 成本极低：比传统支付便宜90%+
- 防通胀：转账销毁50%

## 🔄 完整整合流程

### 场景：俄罗斯买家购买中国商品

```
┌─────────────────────────────────────────────────────────────┐
│                    完整贸易流程                              │
└─────────────────────────────────────────────────────────────┘

【阶段1：发现】
俄罗斯买家 (AIX Box)
    │
    │ ① 开机 → 自动连接LBRTYnet Mesh
    │ ② 搜索商品 (通过Mesh广播)
    │ ③ 发现中国卖家商品列表
    ↓
商品信息通过Mesh传输，消耗 1 CH

【阶段2：下单】
买家决定购买
    │
    │ ④ 创建订单，锁定 Coin Hour 到托管合约
    │ ⑤ 托管合约：买家 + 卖家 + 仲裁 (2-of-3)
    ↓
托管锁定 100 CH

【阶段3：支付确认】
卖家收到订单通知
    │
    │ ⑥ 通过LBRTYnet确认订单
    │ ⑦ 查看托管状态 (链上验证)
    ↓
卖家确认接单

【阶段4：发货】
卖家准备货物
    │
    │ ⑧ 货物打包，信息上链
    │ ⑨ 清关AI处理HS编码
    │ ⑩ 生成清关文件
    ↓
清关AI服务费 5 CH

【阶段5：物流】
货物在途中
    │
    │ ⑪ 每个节点 (AIX Box) 签名确认位置
    │ ⑫ 状态通过Mesh实时同步
    │ ⑬ 买家随时查看
    ↓
物流节点费 10 CH

【阶段6：收货】
货物到达俄罗斯
    │
    │ ⑭ 买家确认收货
    │ ⑮ 托管合约释放 Coin Hour
    │ ⑯ 卖家收到 100 CH
    ↓
交易完成

【阶段7：结算】
卖家可选：
    │
    │ ⑰ CH保留 → 继续使用
    │ ⑱ CH → AIX → 法币
    │ ⑲ CH用于采购
    ↓
循环继续
```

## ⚠️ 关键问题解答

### Q1: 为什么用Coin Hour而不是AIX支付？

**答：**
- Coin Hour自动产生，无需购买
- 小额支付更划算
- 避免AIX价格波动影响

### Q2: LBRTYnet和Skywire什么关系？

**答：**
- Skywire：Skycoin的Mesh网络
- LBRTYnet：独立的Mesh协议
- Skywire Bridge：让Skywire流量走LBRTYnet

### Q3: 网络断了怎么办？

**答：**
- Mesh网络本地互联
- 不需要互联网
- 口岸节点中继

### Q4: 如何保证交易安全？

**答：**
- 托管合约 (2-of-3多签名)
- 仲裁节点
- 链上可追溯

### Q5: 清关AI如何工作？

**答：**
- 商品信息输入
- 自动匹配HS编码
- 生成清关文件
- 合规检查
