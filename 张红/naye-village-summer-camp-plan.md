## 核心闭环

```
收费5000元 → 送等值AIX Token → Coin Hour消费 → 内循环
```

## 收费设计

| 项目 | 内容 |
|------|------|
| 营费 | 5,000元/人 |
| 赠送 | 等值5,000元的AIX Token |
| 村内消费 | 全部用Coin Hour结算 |

**经济学原理：**
- 学员付5000元现金
- 获得5000元AIX Token
- AIX Token产出Coin Hour
- 在村里消费用Coin Hour
- 消费完毕后AIX Token仍保留
- 离开时可带走或继续持有
