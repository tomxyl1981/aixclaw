## Token不是代币，是词元

**词元（Token）** = 建立AIX Claw的元素

词元是：
- 世界的基本构成单位
- 不是投机工具
- 不是交易媒介
- 是**价值生成的物理基础**

---

## 持有 = 长期价值证明

传统代币逻辑：
```
买入 → 等待升值 → 卖出赚钱
```

AIX Token逻辑：
```
买入/贡献 → 持有 → 产生Coin Hour → 消费内循环产品
```

**不是为了卖掉赚钱，而是为了产生Coin Hour后可以买到的东西。**
