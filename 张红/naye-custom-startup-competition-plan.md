## 运作模式

### 1. 项目入驻

```
创业者带想法来 → 那耶村评估 → 选定项目 → 定制孵化 → 参赛
```

**入驻门槛**：
- 项目有创新性（AI、数字主权、乡村经济）
- 创业者愿意用Coin Hour支付孵化费用
- 项目成果那耶村可分享（Skill资产化）

### 2. 定制孵化服务

| 服务 | 内容 | Coin Hour定价 |
|------|------|-----------------|
| 项目打磨 | 商业计划书优化 | 500-1000 CH |
| 市场调研 | 那耶村数据支撑 | 200-500 CH |
| 技术验证 | AIX Box原型测试 | 1000-2000 CH |
| 路演培训 | 演讲稿+PPT设计 | 300-500 CH |
| 大赛报名 | 陪同参赛 | 500 CH |

### 3. 收益分配

**如果获奖**：

| 奖金分配 | 比例 |
|----------|------|
| 创业者 | 70% |
| 那耶村孵化费 | 20% |
| AIX DAO | 10% |

**一亿Coin Hour目标**：

```
获奖项目奖金：1000万CH
那耶村分成：200万CH
获奖项目10个 × 200万CH = 2000万CH
...
累计获奖100个项目 → 一亿CH达成
```
