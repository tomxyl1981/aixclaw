## 二、选果标准

### ✅ 适合做果酱
- **9成熟或过熟**（比做果干更熟）
- 软烂果也可用（不浪费）
- 香味浓郁
- 无霉变、无虫蛀

### ❌ 不适合
- 未成熟青果（酸涩）
- 发霉、腐烂果

## 三、制作流程（6步法）

### Step 1：预处理（15分钟）

```
鲜果 → 清洗 → 去皮（可选）→ 切碎/打浆
```

**去皮方法**：
- 方法A：开水烫10秒 → 冰水激 → 皮自然脱落
- 方法B：用勺子挖出果肉（保留果皮做茶）

**果肉处理**：
| 方法 | 口感 | 适合 |
|------|------|------|
| 切小块 | 有果肉颗粒感 | 涂面包 |
| 打浆机打碎 | 细腻顺滑 | 拌酸奶 |
| 手压成泥 | 自然粗质感 | 传统风味 |

**建议**：新手用切小块，容易控制

---

### Step 2：熬煮（40-60分钟）

```
锅 → 加100ml水 → 倒入果肉 → 中火煮沸 → 加糖
```

**加糖时机**：
- 方法A：果肉煮沸后加糖（推荐，不易糊底）
- 方法B：果肉+糖一起煮（更快出汁）

**火候控制**：

| 阶段 | 时间 | 火候 | 状态 |
|------|------|------|------|
| 初期 | 10分钟 | 中火 | 大量出水，沸腾 |
| 中期 | 20-30分钟 | 小火 | 汁液浓缩，冒大泡 |
| 后期 | 10-15分钟 | 小火 | 粘稠，冒小泡 |

**关键动作**：
- **频繁搅拌**（每2-3分钟），防止糊底
- 撇去表面浮沫（更清澈）

---

### Step 3：加柠檬汁（最后10分钟）

```
果酱浓稠后 → 加柠檬汁 → 搅拌均匀 → 继续熬煮5-10分钟
```

**柠檬作用**：
- 调节酸度，延长保质
- 增强果胶凝胶效果
- 防止氧化变色

---

### Step 4：判断终点

**方法A：温度法**
- 糖度计测：可溶性固形物≥65%
- 温度计测：103-105°C

**方法B：滴落法（推荐）**
- 用勺子舀起果酱，滴落观察
- 滴落时**连成线**而非水珠 → 即可
- 或滴在冷盘上，冷却后不流动 → 即可

**方法C：冷藏法**
- 滴一小勺在冷藏过的盘子上
- 冷却2分钟 → 不流动、表面不粘 → 即可

---

### Step 5：灌装（10分钟）

```
玻璃瓶 → 沸水煮沸10分钟消毒 → 沥干
    → 趁热灌装（85°C以上）→ 拧紧瓶盖
    → 倒置冷却（形成真空）
```

**要点**：
- 果酱温度≥85°C灌装（杀菌）
- 留1cm顶空
- 立即拧紧瓶盖
- 倒置冷却至室温

---

### Step 6：储存

| 储存方式 | 保质期 |
|----------|--------|
| 未开封，阴凉处 | 6-12个月 |
| 开封后，冷藏 | 1-2个月 |
| 糖度≥65% | 更持久 |

## 四、常见问题

### Q1：果酱太稀怎么办？
**原因**：熬煮时间不够或糖度不足  
**对策**：继续小火熬煮，蒸发水分

### Q2：果酱太甜怎么办？
**原因**：糖放太多  
**对策**：下次减少糖量，或加柠檬汁调节

### Q3：果酱结晶怎么办？
**原因**：糖未完全溶解  
**对策**：加少量水，重新加热融化

### Q4：果酱发霉怎么办？
**原因**：瓶子不洁净或糖度不够  
**对策**：严格消毒，提高糖度至≥65%

### Q5：颜色变暗怎么办？
**原因**：氧化  
**对策**：加更多柠檬汁，或缩短熬煮时间
