## 一、野茶的核心价值

### 为什么野茶比茶园茶值钱？

| 维度 | 茶园茶 | 野茶 |
|------|--------|------|
| 树龄 | 10-30年 | 50-500年+ |
| 生长方式 | 人工施肥打药 | 自然生长，零农残 |
| 产量 | 亩产200-400斤 | 亩产20-50斤（稀缺） |
| 市场价 | 50-300元/斤 | 300-5000元/斤 |
| 故事性 | 平淡 | "千年古树""壮族秘境" |

**核心溢价**：稀缺性 + 故事性 + 零农残

### 那耶村野茶的独有优势

| 优势 | 说明 |
|------|------|
| 海拔660-1180米 | 高山云雾茶品质 |
| 千年壮族古村 | 文化底蕴深厚 |
| 梯田景观 | "云上稻乡"意境 |
| 零工业污染 | 纯净产区背书 |
| 梁越IP | 《隐没的战象》文化加持 |

## 五、Coin Hour闭环

```
野茶采摘/制作 → 村民劳动收入Coin Hour
        ↓
游客茶体验 → 支付Coin Hour
        ↓
茶产品销售 → Coin Hour结算
        ↓
村民Coin Hour → 村内消费
        ↓
茶叶收益 → 村集体分红
```

### DAO茶砖模式

```
投资者购买DAO茶砖 → 获得茶叶实物 + 链上确权
        ↓
茶叶存入那耶村茶仓 → 专业仓储（每年增值）
        ↓
投资者可：提货品饮 / 继续存放增值 / 二级市场转让
        ↓
Coin Hour结算仓储费（5 CH/饼/年）
```
