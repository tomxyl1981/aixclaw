# 无花果加工全流程指南

**创建时间**：2026-05-24  
**需求方**：Jane

---

## 一、无花果加工总览

```
鲜果采摘
    ├─→ 初加工（果干、果酱、果脯）← 门槛最低，优先启动
    ├─→ 深加工（酵素、果酒、果醋）← 中等门槛
    ├─→ 提取加工（花青素、果粉、精华）← 需专业设备
    └─→ 化妆品原料（果汁、果油、果酶）← 需OEM合作
```

---

## 二、初加工（优先启动）

### 1. 无花果干

**工艺流程**：
```
鲜果选果 → 清洗 → 去皮（可选）→ 切半/整果
    → 烘干（50-65°C，12-24小时）
    → 回软（密封放置24小时）
    → 分级包装 → 入库
```

**关键参数**：
| 参数 | 要求 |
|------|------|
| 原料成熟度 | 8-9成熟，不宜过熟 |
| 烘干温度 | 50-65°C（先低后高） |
| 烘干时间 | 12-24小时 |
| 含水量 | ≤20% |
| 成品率 | 5-6斤鲜果→1斤果干 |

**设备投资**：
| 设备 | 价格 | 产能 |
|------|------|------|
| 小型烘干房 | 3000-8000元 | 50-100斤/批 |
| 中型烘干机 | 1-3万 | 200-500斤/批 |
| 冻干机（高端） | 5-15万 | 50-100斤/批 |

**成本估算**：
| 项目 | 成本/斤 |
|------|---------|
| 鲜果原料 | 5-6斤×3元 = 15-18元 |
| 电费 | 2-3元 |
| 人工 | 2-3元 |
| 包装 | 3-5元 |
| **总成本** | **22-29元/斤** |
| **市场售价** | **60-120元/斤** |
| **毛利率** | **55-75%** |

---

### 2. 无花果酱

**工艺流程**：
```
鲜果选果 → 清洗 → 去皮 → 切碎/打浆
    → 加糖熬煮（果:糖=3:1）
    → 加柠檬汁调酸
    → 浓缩至可溶性固形物≥65%
    → 热灌装 → 杀菌（100°C，10分钟）
    → 冷却 → 贴标 → 入库
```

**关键参数**：
| 参数 | 要求 |
|------|------|
| 原料成熟度 | 9成熟，软烂果也可用 |
| 糖度 | 果:糖=3:1~4:1 |
| 熬煮时间 | 30-60分钟 |
| 杀菌温度 | 100°C，10分钟 |
| 成品率 | 2-3斤鲜果→1瓶(250g) |

**设备投资**：
| 设备 | 价格 | 产能 |
|------|------|------|
| 夹层锅+灌装线 | 5000-2万 | 100-300瓶/批 |
| 小型果酱生产线 | 3-8万 | 500-1000瓶/批 |

**成本估算**：
| 项目 | 成本/瓶(250g) |
|------|--------------|
| 鲜果原料 | 2-3斤×3元 = 6-9元 |
| 白糖 | 1-2元 |
| 包装（瓶+盖+标） | 2-3元 |
| 人工 | 1-2元 |
| **总成本** | **10-16元/瓶** |
| **市场售价** | **25-50元/瓶** |
| **毛利率** | **55-70%** |

---

### 3. 无花果脯/蜜饯

**工艺流程**：
```
鲜果选果 → 清洗 → 去皮 → 整果/切半
    → 糖渍（30%糖液浸泡24小时）
    → 逐步加糖（每次提高10%，共3次）
    → 沥干 → 烘干（50-60°C，8-12小时）
    → 表面撒糖粉（可选）
    → 包装 → 入库
```

**关键参数**：
| 参数 | 要求 |
|------|------|
| 糖渍总时间 | 3-5天 |
| 终点糖度 | ≥65% |
| 烘干温度 | 50-60°C |
| 含水量 | 18-22% |

**成本估算**：
| 项目 | 成本/斤 |
|------|---------|
| 鲜果+糖 | 12-15元 |
| 人工 | 3-4元 |
| 包装 | 3-5元 |
| **总成本** | **18-24元/斤** |
| **市场售价** | **40-80元/斤** |

## 三、深加工（中等门槛）

### 4. 无花果酵素

**工艺流程**：
```
鲜果清洗 → 切片 → 加糖（果:糖=1:1）
    → 密封发酵（6-12个月）
    → 过滤 → 二次发酵（2-3个月）
    → 灌装 → 杀菌 → 贴标
```

**关键参数**：
| 参数 | 要求 |
|------|------|
| 发酵温度 | 20-30°C |
| 发酵时间 | 6-12个月 |
| 糖度 | 果:糖=1:1 |
| pH | 3.5-4.0 |
| 成品率 | 3-4斤鲜果→1瓶(500ml) |

**设备投资**：
| 设备 | 价格 |
|------|------|
| 发酵罐（玻璃/不锈钢） | 5000-2万 |
| 过滤设备 | 3000-1万 |
| 灌装线 | 5000-2万 |

**成本估算**：
| 项目 | 成本/瓶(500ml) |
|------|---------------|
| 原料 | 15-20元 |
| 糖 | 5-8元 |
| 包装 | 5-8元 |
| 时间成本（6-12月） | 10-15元 |
| **总成本** | **35-51元/瓶** |
| **市场售价** | **100-300元/瓶** |
| **毛利率** | **65-85%** |

---

### 5. 无花果酒

**工艺流程**：
```
鲜果清洗 → 去皮打浆 → 加糖调糖度至22%
    → 接种酵母 → 主发酵（25°C，7-10天）
    → 分离酒液 → 后发酵（15°C，1-2月）
    → 澄清 → 陈酿（3-6月）
    → 过滤 → 灌装 → 杀菌
```

**关键参数**：
| 参数 | 要求 |
|------|------|
| 初始糖度 | 22% |
| 发酵温度 | 主发酵25°C，后发酵15°C |
| 酒精度 | 10-14% |
| 陈酿时间 | 3-6个月（最低） |
| 成品率 | 3-4斤鲜果→1瓶(750ml) |

**成本估算**：
| 项目 | 成本/瓶(750ml) |
|------|---------------|
| 原料 | 10-15元 |
| 酵母+辅料 | 2-3元 |
| 包装（瓶+塞+标） | 5-10元 |
| **总成本** | **17-28元/瓶** |
| **市场售价** | **80-200元/瓶** |
| **毛利率** | **75-85%** |

---

### 6. 无花果醋

**工艺流程**：
```
果酒（同上）→ 接种醋酸菌
    → 醋酸发酵（30°C，1-2月）
    → 过滤 → 陈酿（1-3月）
    → 调配（加水稀释至5-6%酸度）
    → 灌装 → 杀菌
```

**成本估算**：20-30元/瓶(500ml)，售价60-150元

## 四、提取加工（需专业设备）

### 7. 无花果果粉

**工艺流程**：
```
鲜果打浆 → 浓缩 → 喷雾干燥/冻干
    → 粉碎 → 过筛 → 包装
```

**设备**：喷雾干燥机（5-20万）或冻干机（5-15万）

**应用**：奶茶原料、烘焙添加剂、固体饮料

### 8. 花青素提取物

**工艺流程**：
```
果皮/果实 → 乙醇提取 → 浓缩 → 纯化
    → 冷冻干燥 → 花青素粉末
```

**设备**：提取罐+浓缩设备（10-30万）

**应用**：保健食品原料、化妆品原料

## 六、场地与资质

| 产品 | 场地 | 面积 |
