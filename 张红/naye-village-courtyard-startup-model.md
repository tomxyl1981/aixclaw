## 核心定位

```
那耶村 = 创业者试验场 + AI市场辐射器
```

不是简单的民宿，而是**生意孵化器**：
- 院子 = 创业空间（不只是住宿）
- AI工具 = 市场放大器（直播、营销、渠道）
- Skill = 方法资产化（成功经验可复制）

---

## 运作逻辑

```
创业者入驻院子 → 开发生意 → AI工具放大 → 成功模式 → Skill资产化
                                            ↓
                                   验证数据留存 → 后来者复用
```

**关键点：AI不只是辅助，是辐射放大器**

| 传统民宿 | 那耶院子创业 |
|----------|--------------|
| 客人消费 | 创业者生产 |
| 离开就结束 | 离开后资产留下 |
| 无杠杆 | AI放大到全国市场 |

---

## 创业者画像

**目标人群**：
- 想验证商业想法但没场地的人
- 数字游民 + 创业野心
- 想借AI工具降本增效的人

**典型场景**：

| 创业者 | 院子生意 | AI辐射方式 |
|--------|----------|------------|
| 直播主播 | 山景直播间 | 多平台分发+AI剪辑 |
| 手工艺人 | 工作室体验 | AI营销+电商上架 |
| 内容创作者 | 影音工作室 | AI生成+全网分发 |
| 茶饮品牌 | 茶空间原型 | AI配方+标准化输出 |
| 农产品主播 | 田边直播 | AI溯源+供应链管理 |

## AI工具栈（院子标配）

每个院子标配一套AI工具：

```
AIX Box（边缘节点）
├─ 推理能力：7B本地模型
├─ 存储能力：经营数据本地留存
├─ 结算能力：Coin Hour收款
└─ 广告能力：被动收入

AI工具包
├─ 直播工具：多平台推流+AI剪辑
├─ 营销工具：内容生成+自动分发
├─ 电商工具：上架+客服+物流对接
├─ 数据工具：客流+转化+复购分析
└─ Skill生成：成功模式一键资产化
```

**创业者只需要**：
1. 带想法来
2. 用院子验证
3. AI工具放大
4. 成功后带走Skill资产
