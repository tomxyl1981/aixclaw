## 什么是概率合约

**概率合约不是"期货商品"，而是将事件发生概率市场化的金融工具。**

### 核心特征

| 维度 | 期货 | 概率合约 |
|------|------|----------|
| 标的物 | 实物/金融资产 | 事件结果 |
| 价格含义 | 资产价格 | 事件概率 |
| 结算方式 | 实物交割/现金 | 是/否二元结算 |
| 用途 | 套期保值/投机 | 信息聚合/预测 |

### Kalshi例子

```
合约问题："比特币能否破10万？"
合约价格：60美分
市场含义：群体共识认为概率是60%
```

**价格波动反映的是群体对未来的集体判断。**
**交易的不是货物，是人类对世界的认知。**

### 二、概率合约的Coin Hour机制

#### 合约定价

```
事件合约 = X Coin Hour
价格范围：0-100 CH
0 CH = 事件不可能发生
100 CH = 事件必然发生
50 CH = 五五开
```

#### 交易流程

```
用户A买入"是"合约 → 60 CH
用户B买入"否"合约 → 40 CH
总池：100 CH

事件发生：
  用户A获得100 CH（盈利40 CH）
  用户B获得0 CH（亏损40 CH）

事件不发生：
  用户A获得0 CH（亏损60 CH）
  用户B获得100 CH（盈利60 CH）
```

### 三、信息聚合价值

概率合约的核心价值：**聚合分散信息**

#### 传统预测问题
- 专家意见分散
- 信息不对称
- 没有激励机制让知情者分享

#### 概率合约解决
- 知情者有动力买入正确方向
- 价格实时反映信息
- 群体智慧聚合

#### 那耶村例子

```
合约："那耶村音乐节是否盈利？"

村民（内部信息）："听说赞助商已确认，大概率盈利"
  → 村民买入"是"合约，推高价格

投资者（外部信息）："看天气预报，音乐节当天有雨"
  → 投资者买入"否"合约，压低价格

最终价格 = 内部信息 + 外部信息 + 群体判断
```

- 1000万人拥有最先进的AI
- AI能提前预测事件结果
- 信息不对称 → 少数人获利

```
第零世界：AI预测 → 内部获利 → 不扩散

```

当有人用AI预测买入合约，价格立即反映，所有人都能看到。

**信息透明化对抗信息垄断。**
